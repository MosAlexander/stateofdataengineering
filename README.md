# State of Data Engineering 2026

An interactive Streamlit dashboard exploring the 2026 data engineering landscape. Analyze survey responses across personas, tech stacks, AI adoption, organizational challenges, pain points, and skills gaps.

## Demo App

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://stateofdataengineering.streamlit.app/)

## Features
- **Persona Analysis**: Find your tribe, regional benchmarks, solo vs team comparisons
- **Tech Stack Explorer**: Data architecture trends and tool adoption
- **AI Insights**: AI paradox analysis and skeptic profiles  
- **Organization**: Leadership gaps, manager awareness, cost consciousness
- **Pain Points**: Interactive explorer and firefighting predictor
- **Skills**: Education alignment and semantic layer adoption gaps

## Data Source
Survey data from [Joe Reis's Practical Data Engineering Survey](https://joereis.github.io/practical_data_data_eng_survey/).

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
